Flop.js

Scratch rendering-style library for javascript.

Description

Flop.js is a two-part libray. flop-render, and flop-vm. Flop-vm processes sprite positions, and sends them along to flop-render. You use a js syntax with scratch commands, with sprite details in variables above. Go to Getting started to begin!
